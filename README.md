# Power BI Sales Analytics Dashboard

## 📊 Overview
This project analyzes sales performance using Power BI, SQL, and Excel.

## 🛠 Tools Used
- Power BI
- SQL
- Excel

## 📈 Key Insights
- South region generates highest revenue
- Electronics category contributes most sales
- Laptop products have highest profit margins

## 📂 Project Files
- Dataset (Excel)
- SQL Queries
- Power BI Dashboard (.pbix)

## 🚀 How to Use
1. Open the .pbix file in Power BI
2. Explore dashboard using filters
